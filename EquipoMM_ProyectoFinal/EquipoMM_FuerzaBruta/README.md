# Fuerza Bruta, Minimax

Programa con finalidad de implementar el algoritmo de **(Minimax)** dentro del juego de 21.

# Instrucciones de Instalación

Se requiere de un intérprete de la siguiente versión para que el programa funcione correctamente:
- Python 3.13 (de preferencia)

**Instalación:**

- Clona el repositorio: 
``` python
https://github.com/NicodobleG/25B-An-lisis-de-Algoritmos---D01---L_Mi-7-9am/tree/5cf7bf7c1e00b31bd6ed483262e7f22e7b3bdb9d/EquipoMM_ProyectoFinal
```

## Ejecución

1. **Modo de ejecución:**
- Navega al directorio: ```cd /EquipoMM_ProyectoFinal/EquipoMM_FuerzaBruta/src```
- Para ejecutar el script principal, usa el siguiente comando: `python 21.py`

2. **Ejemplos de uso**
