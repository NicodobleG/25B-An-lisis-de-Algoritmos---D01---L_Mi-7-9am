# Visualizador de metodos de Ordenamiento

Programa con el objetivo de intentar resolver el problema del viajero (Travelling Salesman Problem).

# Instrucciones de Instalación

Se requiere de un intérprete de la siguiente versión para que el programa funcione correctamente:
- Python 3.10 (o superior)

**Instalación:**

- Clona el repositorio: 
``` python
git clone https://github.com/NicodobleG/25B-An-lisis-de-Algoritmos---D01---L_Mi-7-9am/blob/f288c271b27b6517107f974d55226ce5b90478c5/GARIN_NICOLAS_TSP/README.md
```

## Ejecución

1. **Modo de ejecución:**
- Navega al directorio: ```cd GARIN_NICOLAS_TSP```
- Para ejecutar el script principal, usa el siguiente comando: `python Garín_Nicolás.py`

2. **Instrucciones de uso**
